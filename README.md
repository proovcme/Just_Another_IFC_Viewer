# Just Another IFC Viewer (BIM AI Viewer v8.0)

Легкий и быстрый веб-вьюер для BIM-моделей (IFC), написанный на Vanilla JS с использованием Three.js и ядра web-ifc. Поддерживает загрузку сводных моделей (Federated BIM) и турбо-индексацию инженерных систем.

## Особенности
* **Производительность:** Аппаратное ускорение через WebGL/Three.js и парсинг IFC на базе WASM (`web-ifc`).
* **Мультимодельность:** Загрузка нескольких `.ifc` файлов одновременно с построением единого сводного дерева проекта.
* **Турбо-индексация:** Мгновенный сбор атрибутов "Имя системы" (System Name) и "Семейство/Типоразмер" (Family and Type) из графа свойств (Psets) без зависания UI. Поддержка как IFC2x3, так и IFC4.
* **Инструменты работы с геометрией:** 
  * Интерактивное пространственное дерево (Spatial Structure).
  * Рулетка (измерения ΔX, ΔY, ΔZ и абсолютного расстояния).
  * Инструменты сечения (X/Y плоскости).
  * Режимы отображения: X-Ray (прозрачность), Ортографическая камера.
  * Изоляция инженерных систем в 1 клик.
* **Режим прораба:** WASD-управление камерой от первого лица (Minecraft-style).

## Стек технологий
* **Frontend:** Vanilla JS, HTML5, CSS3
* **3D Engine:** [Three.js](https://threejs.org/)
* **IFC Parser:** [web-ifc](https://github.com/ThatOpen/engine_web-ifc) / web-ifc-three
* **Backend:** PHP (только для сканирования локальной папки с моделями)

## Установка и запуск
Так как проект использует WASM и модули ES6, его нужно запускать через локальный веб-сервер (прямое открытие `index.html` в браузере выдаст CORS-ошибку).

1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/proovcme/Just_Another_IFC_Viewer.git](https://github.com/proovcme/Just_Another_IFC_Viewer.git)
   cd Just_Another_IFC_Viewer
2. Установите зависимости (web-ifc, three.js):

Bash
npm install
Положите ваши .ifc файлы в папку models/.

3. Запустите локальный PHP-сервер (или любой другой, например Live Server в VS Code):

Bash
php -S localhost:8000
Откройте в браузере: http://localhost:8000/index.php
